# MCP Server Installation Links

This document provides quick installation links for all available MCP servers from the [Cursor MCP Directory](https://cursor.com/docs/context/mcp/directory).

## Overview

Model Context Protocol (MCP) servers enhance the capabilities of our SDLC workflows by providing integrations with external services and tools. Click on any install button below to automatically configure the MCP server in Cursor.

## Configuration Reference

After installation, configure credentials and environment variables in `.cursor/mcp.json`. See individual server sections below for detailed configuration examples.

### Core Integration Servers

#### GitHub MCP Server

```json
{
  "mcpServers": {
```
